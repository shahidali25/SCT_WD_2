🧮 Calculator Web Application

This project is a Calculator Web Application developed using HTML, CSS, and JavaScript.
It performs basic arithmetic operations and provides a simple and interactive user interface.

---

🚀 Features

- Perform basic arithmetic operations
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)
- Clear and delete buttons
- Interactive button interface
- Responsive design

---

🛠 Technologies Used

- HTML – Structure of the calculator
- CSS – Styling and layout design
- JavaScript – Calculator logic and functionality
